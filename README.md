# 拔河数学游戏（PNG/WEBP素材版）

## 使用说明
1. 将拔河人物图片放到：
   - `assets/images/tug-strip.webp`（推荐）
   - `assets/images/tug-strip.png`（兜底）
2. 直接打开 `index.html` 即可运行
3. 或使用静态服务部署到 GitHub Pages

## 素材要求（非常关键）
- **透明背景**
- 一张图里包含：左队 + 绳子 + 中间红标 + 右队
- 人物必须握住绳子（手与绳子连接）
- 推荐尺寸：`1600x500` 或接近比例

## GitHub Pages
- 推送代码到仓库
- Settings → Pages → Deploy from branch → main / root
- 保存后即可访问
