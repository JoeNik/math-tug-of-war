# Math Tug of War

适合 5–10 岁儿童的课堂互动数学拔河游戏。

## 功能
- 左右两队答题对抗
- 答对后绳子向自己方向移动
- 适合触摸大屏
- 幼儿模式 / 基础模式 / 进阶模式
- 简单加减乘法
- 分数统计
- 计时器
- 全屏
- 声音开关
- 胜利弹窗
- 可部署到 GitHub Pages / Netlify / Vercel
